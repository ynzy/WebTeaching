## 克隆项目(不用执行了)
```js
git clone https://github.com/ynzy/webTeaching
```
## 打开项目
进入你的项目文件夹
···js
D:\qianduan\webTeaching
···
拖进你的vscode
这个就是你的项目
## 编写文件之后提交文件
打开终端
···js
git add .
```
## 提交描述
就是说，你提交乐文件，给这个提交一个备注，以后好查看
···js
git commit -m '我提交乐一条记录'
···
## 将提交记录推送到远程仓库
前面都是本地操作，最后这个才是提交到GitHub上
···js
git push
```
